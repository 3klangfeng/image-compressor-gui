# 🖼️ Image Compressor GUI

一个基于 Python + Tkinter 的 **高性能图片压缩工具（带 GUI）**  
支持多线程、自动缩放、批量处理，适合日常图片压缩、素材整理。

## ✨ 功能特性

- ✅ 支持 JPG / PNG / WEBP / BMP / GIF / TIFF
- ✅ 自动转 JPG（兼容性最好）
- ✅ 二分法智能压缩，尽量接近目标大小
- ✅ 自动缩放尺寸（可关闭）
- ✅ 多线程并发处理（最高 16 线程）
- ✅ 可选原图备份
- ✅ GUI 可视化界面（ttkbootstrap 美化）

## 🖥️ 运行环境

- Python 3.9+
- Windows / macOS / Linux
